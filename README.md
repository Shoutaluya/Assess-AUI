## 📊 Overview

**A.S.E.S.S** is an advanced, multidimensional diagnostic tool designed for AUI. It utilizes predictive classification models (Random Forest, Support Vector Machine, and Decision Trees) to evaluate student academic trajectories, providing real-time inference on graduation profiles and policy clearance.

## 🧠 Diagnostic Models

The system evaluates data against three primary algorithms to determine the terminal CGPA probability distribution:

*   <strong style="color: #223E77;">Random Forest (RF):</strong> Primary active model. High precision baseline.
*   <strong style="color: #223E77;">Support Vector Machine (SVM):</strong> Secondary validation layer.
*   <strong style="color: #223E77;">Decision Tree (DT):</strong> Structural analysis layer.
